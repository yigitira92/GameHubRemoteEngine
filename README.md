# GameHub Remote Engine

Modern remote desktop manager built for **[Gamehubserver.com.tr](https://gamehubserver.com.tr)** by **Servervia Bilişim Yazılım ve Telekomünikasyon Hizmetleri Ltd. Şti.**

## Downloads

Grab the latest installer from the [Releases](https://github.com/yigitira92/GameHubRemoteEngine/releases) page. The app auto-updates from then on.

## About

This repository hosts the signed update artifacts for the application. Source code is private; only build output is published here for distribution.
